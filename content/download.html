--- title: Download ---

Download

The latest stable releases are:

Core 1.6.0
Optional Modules 1.6.0

To download the latest release 1.6.1 of the core module, SBT users should add:

libraryDependencies += "com.assembla.scala-incubator" % "graph-core_2.10" % "1.6.1"

to your SBT build file.

For Maven users, please set the following ids:

group id com.assembla.scala-incubator
artifact id graph-core_2.10
version 1.6.0

The latest stable releases of the optional modules are:

graph-constrained_2.10 1.6.0
graph-dot_2.10 1.6.0
graph-json_2.9.2 1.6.0
You may also manually donwload the jar and Scaladoc from Sonatype.
