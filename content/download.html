Download

The latest stable releases are:

graph-core 1.7.2
Optional Modules 1.7.0

To download the latest release of the core module, SBT users should add:

libraryDependencies += "com.assembla.scala-incubator" % "graph-core_2.10" % "1.7.2"

to the SBT build file. As a Maven user, please set the following ids:

group id com.assembla.scala-incubator
artifact id graph-core_2.10
version 1.7.2

The latest stable releases of the optional modules are:

graph-constrained_2.10 1.7.0
graph-dot_2.10 1.7.0
graph-json_2.10 1.7.0
You may also manually download the Jars and Scaladoc from Sonatype.

1.7 Release Notes

1.7.2

  • new: method upsert for mutable graphs.
  • fix: due to a regression introduced in 1.7.1 findOutgoingTo and findIncomingFrom didn't work correctly.

1.7.1

  • mod: Several node-level methods and equals/hashCode for edges have been optimized.
  • fix: shortestPathTo GitHub issue #12.
  • fix: method draw.
  • fix: builder pattern imposed by GraphCanBuildFrom.

1.7.0

  • new: Edge extractors have been added/modified. Latter may require user-site source code changes in edge patterns. See Inspecting Endpoints and More Edge Patterns.
  • mod: SBT upgraded to 0.13.0.
